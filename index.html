Live Cooldown Timers

Click once when you see Flash or an ult used. LoL Buddy counts it down with a visual progress bar and alerts you the moment it's back up.

🔥

Ability Haste Aware

Ult timers automatically adjust for each enemy's ability haste so you're never tracking the wrong cooldown length.

🎮

Global Keyboard Shortcuts

Mark spells without alt-tabbing. Numpad 1–5 for D spells, Numpad 6–0 for F spells, Ctrl+Num 1–5 for ults. Works while League is focused.

🎙

Voice Commands

Say "Zed flash" or "Leona ult" and the timer starts automatically — fully offline, powered by on-device Whisper AI.

🧿

Enemy Rune Detection

See each enemy's keystone and secondary rune tree at a glance, colour-coded to match League's own rune palette.

📌

Always on Top

Transparent, frameless overlay that stays above League in full-screen windowed mode. Drag it wherever you want.

Up and running in 60 seconds

  1. 1
    Download & install — run the LoL Buddy Setup installer.
  2. 2
    Launch a game — LoL Buddy detects your match automatically via the League Live Client API.
  3. 3
    Click or press a key — mark an enemy's spell as used and watch the timer count down.
  4. 4
    Get notified — a sound and system alert fires the moment the cooldown expires.

Is it safe to use?

Yes. LoL Buddy only reads data from the official League Live Client API that Riot exposes on your local machine. It never reads game memory, injects code, or communicates with any external server during a match. This is the same API used by approved overlay tools.

Voice processing runs entirely on your device — no audio is ever sent anywhere.
